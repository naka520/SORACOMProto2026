# もの置き診断アプリ 実装計画書

GPSマルチユニットとSORACOM Flux、Azureを活用した、環境データに基づく「もの置き」可否判定システムの設計案です。

---

## 1. システム概要

本アプリは、**「特定の場所に特定の物を置いて大丈夫か」**を、センサーデータ（温湿度）、画像解析（物体認識）、および気象予報を組み合わせてAIが判定するソリューションです。

## 2. システム構成図 (Architecture)

1.  **データ計測:** GPSマルチユニット SORACOM Edition
    - 位置情報、温度、湿度を定期的にSORACOM Harvest Data / Fluxへ送信。
2.  **フロントエンド:** Azure Static Web Apps (TypeScript / React or Vue)
    - ユーザーがスマートフォンで「置きたい物」を撮影。
    - カメラ画像をAzure Blob Storageに保存。
3.  **ロジック・AI:** SORACOM Flux
    - **メイン処理**: GPSマルチユニットからセンサー値（温湿度、位置情報）を取得。
    - フロントエンドから受け取ったカメラ画像を参照。
    - ウェザーニューズAPIから天気予報情報を取得。
    - 上記データを統合してLLM（GPT-4o等）にプロンプトを作成し、「置き場所の適切性」を診断。
4.  **バックエンド:** Azure Functions (TypeScript)
    - SORACOM Fluxからの診断結果（webhook）を受け取り。
    - 診断結果をフロントエンドへAPI経由で提供。
5.  **画像保存:** Azure Blob Storage
    - フロントエンドから送信されたカメラ画像を保存。
    - SORACOM Fluxから画像参照が可能。
6.  **外部データ:** ウェザーニューズAPI
    - 位置情報に基づく屋外の天気予報情報の提供。

---

## 3. 画面設計 (3画面構成)

### ① カメラ撮影画面 (Camera View)

- **機能:**
  - ブラウザのカメラ起動とキャプチャ。
  - 位置情報の取得（ブラウザGPS）。
- **ユーザー体験:** - 「診断する」ボタンで撮影し、画像をAzure経由でSORACOM Fluxへ転送。

### ② 分析中の画面 (Analyzing View)

- **機能:**
  - ポーリングまたはWeb PubSubによるステータス監視。
- **ユーザー体験:**
  - 「環境データを照合中...」「AIが材質と湿度の相性を確認中...」といった動的なテキストを表示し、期待感を醸成。

### ③ 結果画面 (Result View)

- **機能:**
  - SORACOM Fluxから返却された診断結果を表示。
- **表示項目:**
  - **総合判定:** 判定アイコン（◎/○/△/×）とスコア。
  - **AIアドバイス:** 例「この本は紙製でカビやすいため、現在の湿度75%の環境では除湿剤が必要です。」
  - **予報連動:** 「明日は雨の予報です。屋外に置く場合は防水シートを使用してください。」

## 5. SORACOM Flux 処理フロー案

1.  **データ集約:**
    - GPSマルチユニットから最新の温湿度・位置情報を取得。
    - Webアプリから送信された画像の参照情報（Blob URL）と座標を受け取る。
2.  **外部データ連携:**
    - 座標を元にウェザーニューズAPIを叩き、24時間以内の天候(雨予報)を取得。
3.  **AIプロンプト作成:**
    - Blob Storageから画像を取得し、以下を問うプロンプトを作成：
      - 「画像に写っている物は何か？」
      - 「その物の材質は何か？」
      - 「現在の温湿度と明日の天気で劣化のリスクはあるか？」
4.  **アクション出力:**
    - 構造化されたJSON（判定・理由・対策）をAzure FunctionsのWebhookに送信。

---

## 6. 今後の開発ステップ

1.  **Phase 1 (MVP):** - GPSマルチユニットのデータをFluxで受信し、固定画像に対して「温湿度のみ」で判定するロジックを組む。
2.  **Phase 2 (Visual Logic):** - フロントエンドから画像を送信し、物体認識と温湿度を組み合わせた判定を実現。
3.  **Phase 3 (External API):** - 天気予報APIを統合し、屋外判定の精度を高める。
